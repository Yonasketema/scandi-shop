## Installation

Install the server and run .

You can find the endpoint [here](https://github.com/scandiweb/junior-react-endpoint) 
 
1. clone the repo
   ```sh
   git clone https://github.com/Yonasketema/scandi-shop.git
   ```
2. install packages
   ```sh
   yarn install
   ```
3. start client
   ```sh
   yarn start
   ```
